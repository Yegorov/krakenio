Krakenio client
===============

Simple python script, which upload image to [kraken.io](https://kraken.io) and download compress image

## How use

```bash
$ python kraken.py <path/to/image.jpg> <path/to/folder>
```

## Dependices

* [requests](http://docs.python-requests.org/en/master/)
